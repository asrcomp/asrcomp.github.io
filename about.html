Syllabus

Programming Concepts

  • Python programming
  • Functions, modules, debugging
  • Object-oriented programming

Data Structures & Algorithms

  • Stacks, queues, linked lists
  • Trees, hash tables
  • Searching & sorting algorithms

Basic Web Development

  • Web application architecture
  • HTML/CSS basics
  • Flask framework

Networking & Security

  • Network protocols & architecture
  • Threats, encryption & authentication
  • IP & MAC addressing, routing

Databases & SQL

  • Database design & normalization
  • SQL queries & optimization
  • NoSQL basics

Assessment

Exam Requirements

  • Paper 1: Computing Theory
    • 3 hours written paper
    • Covers topics such as Computing Networks, Network Security, Databases, etc
    • 60% weightage
  • Paper 2: Computing Practical
    • 3 hours to complete 4 programming tasks
    • Covers topics such as Python, Data Structures and Algorithms, Object Oriented Programming, SQL and Flask Web Applications, etc
    • 40% weightage
